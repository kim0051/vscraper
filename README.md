A Python web scraping tool that allows users to enter any URL, scrape all files of the type they specify, and then download those files to their local directory.

## Requirements
  - This program was made and is run with Python 3.4.2 and uses these libraries in order to run:
  
      - [requests](http://docs.python-requests.org/en/latest/)
      - [BeautifulSoup4](http://www.crummy.com/software/BeautifulSoup/bs4/doc/)
      - [urllib](https://docs.python.org/3/library/urllib.html)
      - [time](https://docs.python.org/2/library/time.html)
